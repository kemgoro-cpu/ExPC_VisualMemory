# External PC Visual Memory 0.2.2 — 初回起動

## 1. 起動

`visual-memory\visual-memory.exe`を実行します。ローカルUIは先に開き、OCRと意味検索モデルはバックグラウンドで準備されます。黄色い「準備中」表示が消えるまで、記録開始ボタンは無効です。

## 2. 映像入力

キャプチャーボードを接続し、「デバイスを再検出」から映像入力を選択します。一度選んだ入力は次回起動時も記憶されます。

## 3. 記録と検索

「記録を開始」で画面履歴を保存します。検索欄ではOCR文字検索と意味検索を併用できます。「さらに読み込む」で過去の候補を追加表示できます。

## 4. AIへ渡す文書

必要な画面だけを選択し、右側のバスケットからコンテキスト文書を作成します。生の画面履歴はMCPへ公開されません。作成済み文書は24時間後に既定で期限切れになります。

## 5. セキュリティ

画面履歴はローカルに保存されます。APIはローカルホスト限定で、ランダムトークンとCSRFトークンにより保護されます。

## 配布プロファイル

- `lite`: OCR・意味検索モデルを含まない小容量版。画像保存と手動選択を確認する用途です。
- `full`: PaddleOCR、多言語E5、モデルを同梱した完全オフライン版です。
